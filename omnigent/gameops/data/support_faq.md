# Support FAQ

## Missing rewards

Collect player id, server id, event id, reward name, screenshot if available, and the timestamp when the player expected the reward. Check whether the issue is isolated to one account or appears in event delivery metrics.

## Payment succeeded but reward missing

Ask for order id, platform receipt id, server id, player id, and purchase timestamp. Payment cases must be escalated when the receipt is valid but the game ledger has no matching grant.

## Account action requests

Account bans, unbans, item recovery, server transfer, and rollback requests require human review. The agent may summarize policy and collect required evidence, but must not approve account actions.
